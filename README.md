# API_Arduino_IOPort

Code to access [Arduino](https://www.arduino.cc/) interface between MATLAB using [Psychtoobox](http://psychtoolbox.org/) **IOPort** function.

# Install
Clone or download this repo, and add it to MATLAB's path.
